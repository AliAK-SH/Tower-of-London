Locate the div immediately following your h1 and add mt-6 (or mt-8 for more space) to its class list.

Why this happens:

The h1 above it uses bg-clip-text and a large font size (text-5xl). In some browsers, the line-height of clipped text doesn’t push the next element down far enough. Adding mt-8 (margin-top) ensures there is a physical gap between the Persian heading and the English subtitle.
